# System requirements

WinLic release candidates require .NET Framework 4.8. Use x86 on x86 Windows, Windows 10 on Arm, or as fallback; use x64 on native x64. Windows 11 Arm emulation is experimental. AnyCPU is development-only and native ARM64 is unavailable.

Windows 7 SP1/8.1 are EOL technical targets. Windows 8.0 is unsupported. Windows 10 22H2 runs the tested payloads but is EOL. Only Windows 10 x64 build 19045 is runtime-verified in Phase 8. Run `WinLicAudit.Cli.exe compatibility` for local, non-network diagnostics.
