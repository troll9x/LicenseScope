# Changelog

## 1.0.0-rc.1

- Unified read-only Windows, Office, Autodesk, Adobe and SketchUp audit engine.
- WPF Scan All experience and CLI with JSON, CSV and printable HTML reports.
- Sensitive-data masking, scanner isolation and compatibility guards.
- Explicit x86/x64 payloads and offline universal installer.
- ARM64 uses x86 emulation; no native ARM64 claim.
- Windows 7 SP1/8.1 runtime support remains unverified and both platforms are EOL.
