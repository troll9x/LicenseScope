# Upstream

License Scope is a private-development continuation of the WinLic codebase originally published at `https://github.com/ardennguyen/WinLic.git` by Arden Nguyen Duc Huy.

The first reachable upstream commit is `f2633e82845aaf137ffeacab2d1436ab550ca56e`. The last upstream commit before the architecture and rebranding work is `bf8dd86fafb90780dc6c54bf533f16a1664fbfde`.

The user has confirmed permission from the WinLic owner for continued development based on that baseline. The permission record is user-managed and is not stored in this repository. This statement does not claim that WinLic is MIT-licensed and does not supply missing public-distribution terms.

Historical upstream material retained for provenance is isolated under `upstream/` and is not part of the License Scope build, installer, or release payload.

Public source and binary distribution remain deferred pending a separate license and third-party-rights review. See `docs/audit/phase-11-upstream-license-audit.md`.
