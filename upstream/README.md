# Preserved upstream material

This directory contains historical WinLic material and the separated legacy administrative GUI retained for provenance and comparison. It is not referenced by `LicenseScope.sln`, parsed by production CI, copied into installer payloads, or included in release bundles.

The preserved PowerShell implementation and `LegacyApp` files contain administrative and network behavior that is outside the read-only License Scope production boundary. Do not invoke or compile them as part of License Scope operation or validation.
